# Gold Rounded Badge HTML Design

## Goal

Create a standalone HTML/CSS recreation of the supplied cream-gold rounded square badge.

## Output

- File: `docs/art/gold-rounded-badge.html`
- Format: one self-contained HTML document.
- Dependencies: none.

## Visual Requirements

- Center the badge on a light checkerboard background so the transparent-style edges are visible.
- Draw the badge entirely with HTML and CSS. Do not use image files, SVG files, or canvas.
- Use a scalable `--badge-size` custom property for the main square.
- Match the reference with a soft warm cream fill, rounded corners, raised outer rim, inset gold groove, and subtle highlight/shadow layers.

## Verification

- Add a small Node verifier that confirms the expected HTML/CSS structure exists and no external image/script is used.
- Open the HTML in a browser-capable render path and inspect the visual result before handoff.
