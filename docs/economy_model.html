Single source of truth for the curve tuning. Everything below is computed from the dials. Endgame anchor: finishing map 5 = a fixed total-click budget. Time unit derived at 2 clicks/sec.

Dials

Edit any dial → every table recomputes. cpc = clicks per coin (per map). Merge is 2:1, generators pop tier-1, so a tier-N item = 2^(N-1) clicks = 2^(N-1) water.

Master budget

Conversion table (everything → time @ 2 clicks/sec)

Per-tier: build cost + quest reward

Quest exp = round(clicks / clicksPerExp) (effort-based, flat across maps). Quest coins = round(clicks / cpc[map] × depthBonus^(tier-4)) — deeper merges pay slightly better per click. Quests pay no acorns (acorns are milestone/IAP only).

Unlock ladder (the 40 spot gates)

Per-map exp increment inc(z) = UNLOCK_BASE + z×UNLOCK_STEP, scaled so the final spot lands at the total-exp budget (= totalClicks ÷ clicksPerExp).

Level curve (cosmetic badge)

Level cost L1 = levelBaseClicks, ×levelGrowth each level. Level gates nothing now; its only role is the per-level-up reward.

Time spent on each level

Active-play time to clear each level (level clicks ÷ 2 per sec). Bars are coloured by the map you're on when you cross that level — note how many levels fall inside The Farm (front-loaded leveling).
